API - .tableFilter(options)


TableFilter is initialized using the .tableFilter function with an options object:


Options

Type: Object
All settings for the table filter
  • thead (default: null)
    Type: HTML or JQuery object representing a thead
    Description: If the thead where the filters should be added is not in the table that is filtered (could be the case when using a fixed header) then this should be the HTML or JQuery wrapped thead object
  • columnFilters (default: empty array)
    Type: Array of ColumnFilter objects
    Description: The ColumnFilter objects defines each column in the table and the filter controls that should be used
  • additionalFilters (default: empty array)
    Type: Array of AdditionalFilter objects
    Description: This is should be an array of AdditionalFilter objects if filters outside the of the columns should be used. See more information about the AdditionalFilter object below.
  • clearFilterControls (default: empty array)
    Type: Array of HTML or JQuery objects
    Description: These HTML or JQuery objects will clear the filters when they are clicked.
  • filterDelay (default: 200)
    Type: integer
    Description: The delay between the last filter expression change and when the filtering starts. Given in number of milliseconds.
  • enableCookies (default: true)
    Type: boolean
    Description: Filter expressions will be saved in a cookie and persisted over postbacks when this is set to true.
  • filteringRows (default: function() {})
    Type: function, void function(FilterStates)
    Description: Called before filering starts with information about the filters. See FilterStates below.
  • filteredRows (default: function() {})
    Type: function, void function(FilterStates, Statistics)
    Description: Called after filtering has finished with information about filters and the number of rows shows. See FilterStates and Statistics below.
  • shouldFilterRow (default: null)
    Type: function, boolean function(Integer, picnet.ui.filter.TableFilterRow)
    Description: If set, this function is called for each row in the table to determine if filtering should be applied to it or not. The first argument is the index of the row, and the second one is FXTableFilters internal data representation of the row.

ColumnFilter

Type: Object
The filter and filtercontrol settings for a column in the table
  • columnName (default: null)
    Type: String
    Description: A name for the column that is a valid JavaScript property name
  • toolTipMessage (default: null)
    Type: String
    Description: The tooltip that should be used for the filter control.
  • defaultText (default: null)
    Type: String
    Description: Default string that should be set in the filter control
  • onlyText (default: false)
    Type: Boolean
    Description: If set to true then only the text (without the HTML) is extracted from the table cells. There is a performance penalty to doing this and it therefore defaults to false
  • includeData (default: true)
    Type: Boolean
    Description: Add the data of this column to the internal representation of the row (picnet.ui.filter.TableFilterRow), that is used when performing the filtering. If the column is not used in any filtering expression then you can set this property to false to improve performance and reduce memory consumption.
  • filterClass (default: null)
    Type: String
    Description: The css class that is added to the <td> tag created in the header for the filter control.
  • colSpan (default: null)
    Type: Integer
    Description: The colspan to use on the <td> tag created in the header for the filter control.
  • createHeaderFilterCell (default: true)
    Type: Boolean
    Description: Set to true if a <td> tag shouldn't be created in the header for this column. Typically used when a previous column used colSpan larger than 1.
  • inputType (default: text)
    Type: String
    Description: The possible values: 'text', 'ddl', 'custom' or 'empty'. When set to null or 'text' a textbox will be used as filter control, when set to 'ddl' then a dropdown list will be used. When set to 'custom' you will have to set the control in the control property and supply functions for getValue, setValue and setupTrigger. When set to 'empty' then no filter control will be created.
  • getValue (default: null)
    Type: function, String function()
    Description: Function without arguments that returns the value of the filter control. This is used only when inputType is set to 'custom' and must be supplied then.
  • setValue (default: null)
    Type: function, function(String)
    Description: Function used to set a string value in the filter control. This is used only when inputType is set to 'custom' and must be supplied then.
  • setupTrigger (default: null)
    Type: function, function(func)
    Description: Function used to setup the triggering of filtering. Set up your custom filter control to call the function supplied as argument when it changes. This is used only when inputType is set to 'custom' and must be supplied then if filtering should happen automatically.
  • matchFunction (default: null)
    Type: function, Boolean function(ctrl, picnet.ui.filter.TableFilterRow, textFilterValue, normalisedTokens)
    Description: Function that determines if the row matches the custom filter control. First argument is the filter control, second is the internal representation of the row, third the value from the filter control and the fourth is a normalised post fix array of token strings created from the the filter value (third parameter). This is used only when inputType is set to 'custom' and must be supplied then.
  • control (default: null)
    Type: HTML or JQuery object
    Description: The control object used to filter this column
  • defaultTextColor (default: null)
  • normalTextColor (default: null)

FilterStates

Type: Object, picnet.ui.filter.FilterState
The filter and filtercontrol settings for additional filter controls that live outside the table
  • id
    Type: String
    Description: The HTML object id of the filter control.
  • columnIndex
    Type: Integer
    Description: The index of the column filter. If it is an additional filter then this is a negative value.
  • value
    Type: String
    Description: The value of the filter control.
  • type
    Type: String
    Description: This is the inputType (see ColumnFilter above). Can be 'text', 'ddl' or 'custom'.

Statistics

Type: Object
The filter and filtercontrol settings for additional filter controls that live outside the table
  • totalNumberOfRows
    Type: Integer
    Description: The total number of rows in the table
  • numberOfRowsVisible
    Type: Integer
    Description: Number of rows that are visible with the current filtering

picnet.ui.filter.TableFilterRow

Type: Object
The internal representation of a table row. This object is sent to the shouldFilterRow function if one is set up. Can be used for custom filtering logic.
  • _isMatch (default: true)
    Type: bool
    Description: Used to indicate if the row matches the filters
  • _allText
    Type: String
    Description: A concatenation of the text in each HTML cell, with a tabcharacter as separator. If onlyText is set to true for the column, then only the text of the cell is used. Otherwise all the content, inclusing HTML code.
  • _tr
    Type: tr HTML object
    Description: The HTML table row object
