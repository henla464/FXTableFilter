FX Table Filter: Demo

Filter through the cells by typing in the input fields at the top of the columns or by using the dropdown lists.

This demo demonstrates how you can filter correctly on columns that contains HTML tags. "Text Column 1" and "Number Column" both contains HTML links in the table cells. For the filtering to only filter using the text of these columns (and not the HTML tags) you must set the "onlyText" option to true for these columns. This option defaults to false due to performance reasons. If this option had not been set on "Number Column" you could have typed for example "href" in its filter and it would have matched all rows.
