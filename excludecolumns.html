Description

This example shows how to have columns without filter controls in the header. The second column can still be search using the quick find filter. The values in the columns are saved internally and if you use a custom filter function they will be available. The third column however can not be search for using the quick filter, the values are not saved internally and will not be available to custom filter functions. There can be performance benefits from excluding columns from being searchable.

Code

  • JavaScript
  • HTML
  • CSS
           $(document).ready(function () {
             var tooltip = 'Your tooltip here!';
             var optionsexcludecolumns = {
               columnFilters: [ 	
            	  { columnName: 'col1', inputType: 'text', toolTipMessage: tooltip},
            	  { columnName: 'col2', inputType: 'empty'},
            	  { columnName: 'col3', inputType: 'empty', includeData: false},
               ],
               additionalFilters: [
                  { control: $('#quickfind'), inputType: 'text', defaultText: 'Type to filter on columns 1 and 2', defaultTextColor: 'red', normalTextColor: 'blue'}
               ]
            };

             $('#demotable-excludecolumns').tableFilter(optionsexcludecolumns);
           });
