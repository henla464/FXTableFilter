Description

The table can be filtered on each column by typing in the textboxes in the table header. A search string will match a text if the text contains the search string. It is also possible to use search expressions, see below.

Search expressions

Quotes (\") match phrases. (not) excludes a match from the results. (or) can be used to do Or searches. I.e. [red or blue] will match either red or blue. Numeric values support >=, >, <=, <, = and != operators.
