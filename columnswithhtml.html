Description

This example demonstrates how you can filter correctly on columns that contains HTML tags. "Text Column 1" and "Number Column" both contains HTML links in the table cells. For the filtering to only filter using the text of these columns (and not the HTML tags) you must set the "onlyText" option to true for these columns. This option defaults to false due to performance reasons. If this option had not been set on "Number Column" you could have typed for example "href" in its filter and it would have matched all rows.

Code

  • JavaScript
  • HTML
  • CSS
           $(document).ready(function () {
             var tooltip = 'Your tooltip here!';
             var optionscolumnswithhtml = {
            	columnFilters: [ 	
                  { columnName: 'col1', inputType: 'ddl', toolTipMessage: tooltip, onlyText: true},
                  { columnName: 'col2', inputType: 'text', toolTipMessage: tooltip, onlyText: true},
                  { columnName: 'col3', inputType: 'text'}
                ]
             };

             $('#demotable-withhtml').tableFilter(optionscolumnswithhtml);
           });
