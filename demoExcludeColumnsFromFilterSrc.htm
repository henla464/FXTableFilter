FX Table Filter: Demo

This demo shows how to have columns without an input control in the header. The second column can still be search using the quick find filter. The values in the columns are saved internally and if you use a custom filter function they will be available. The third column however can not be search for using the quick filter, the values are not saved internally and will not be available to custom filter functions. There can be performance benefits from excluding columns from being searchable.
