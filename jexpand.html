Description

This example demonstrates how you can use the JExpand JQuery plugin to make every second row expandable.

Explanation of the options

shouldFilterRow: Function that returns true if the row should checked agains the filter. Rows for which this function returns false is not touched.
filteredRows: This function is called after filtering of rows are done. It is used here to hide the expandable row when its parent row is hidden by the filter.

See Expand table rows with jQuery - jExpand plugin for info about the JExpand plugin

Code

  • JavaScript
  • HTML
  • CSS
           $(document).ready(function () {
             var tooltip = 'Your tooltip here!';
             $("#demotable-jexpand").jExpand();

             var optionsjexpand = {
               columnFilters: [ 	
                   { columnName: 'Country', inputType: 'text', toolTipMessage: tooltip},
                   { columnName: 'Population', inputType: 'text', toolTipMessage: tooltip},
                   { columnName: 'Area', inputType: 'text'},
                   { columnName: 'Official languages', inputType: 'ddl', selectOptionLabel: 'Select...'},
                   { columnName: 'col5', inputType: 'empty'}
             	],
                shouldFilterRow: function(rowIndex, tr) { return (rowIndex % 2 !== 0); },
                filteredRows: function(filterStates, stats) {
                     $("#demotable-jexpand").find("tr.odd").each(function( index ) {
                        if (!$(this).is(":visible")) {
                           $(this).next().hide();
                        }
                     });
                }
             };

             $('#demotable-jexpand').tableFilter(optionsjexpand);
           });
